# CredZo — Instant UPI Credit Line

Interactive mobile prototype for the CredZo instant UPI credit line experience.

## Local development

```bash
npm install
npm run dev
```

Open [http://localhost:3000](http://localhost:3000).

## GitHub Pages

The app is deployed automatically on push to `main` via GitHub Actions.

Live site: `https://<your-username>.github.io/credzo-app/`
